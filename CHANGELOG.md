# Changelog

## v1.1.0 — 2026-07-17 (M5: Post-Launch-Features)

- **Externe Links** in der Vorschau öffnen in einem neuen Tab (`target=_blank`, `rel=noopener`).
- **Suche überarbeitet:** Live-Trefferliste unter dem Suchfeld (nicht mehr nur der oberste
  Treffer); Highlight über **Titel + Vorschau**; **Treffer-zu-Treffer-Navigator** (◀ n/N ▶);
  **Filter** nach Bereich (Notizen) bzw. Segment (System-Ring).
- **Zurück/Vorwärts-Navigation** durch die geöffneten Dokumente (In-Memory-History).
- **Resizable Inspektor-Panel** per Zieh-Griff; Breite wird lokal gemerkt.
- **CI:** GitHub Actions — pytest auf Linux/Windows/macOS (Python 3.11 + 3.12) + Frontend-Build.
